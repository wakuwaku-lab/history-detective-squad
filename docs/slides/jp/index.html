プロジェクト概要

歴史探偵団は中国歴史をテーマにした教育アドベンチャーゲームです。プレイヤーは若い「歴史探偵」になり、様々な歴史時代を旅しながら、パズル謎解き、歴史人物カードコレクションを通じて、真の歴史マスターを目指します。

歴史探偵団 教育アドベンチャーゲーム

主な機能

🗺️

歴史タイムマップ

  • 6つの歴史時代：先秦、秦漢、三国、隋唐、宋元、明清
  • ミスト解除システム - 探偵任務をクリアして新エリアを開放
  • 精美な中国風ビジュアルデザイン
🔍

探偵任務システム

  • 各時代に複数の探偵任務
  • 歴史人物・重大事件・文化功績に関する問題
  • 難易度：簡単・普通・難しい
👤

探偵育成システム

  • レベル成長：見習い → 初級 → 上級 → マスター → レジェンド探偵
  • 経験値獲得 - 事件解決でポイントを入手
  • 実績システム - 限定バッジを解除
👥

歴史人物図鑑

  • 各時代の歴史上の人物をコレクション
  • 彼らの功績と貢献を学ぶ
  • スキルタグで人物の特徴を表示
