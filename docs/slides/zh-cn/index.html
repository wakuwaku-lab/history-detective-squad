Author: exia.huang Project: history-detective-squad

项目简介

历史奇探团 是一个以中国历史为主题的教育冒险游戏。玩家扮演一名年轻的"历史小探"，穿越不同的历史时期，通过解谜、答题、收集历史人物卡牌，成为真正的历史大师。

历史奇探团 教育冒险游戏

特色功能

🗺️

历史时光地图

  • 六大历史时期：先秦、秦汉、三国、隋唐、宋元、明清
  • 迷雾解锁机制，通过完成探案任务逐步解锁新区域
  • 精美的中国风视觉设计
🔍

探案任务系统

  • 每个时代都有多个探案任务
  • 题目涵盖历史人物、重大事件、文化成就
  • 难度分级：简单、中等、困难
👤

侦探养成系统

  • 等级成长：见习侦探 → 初级侦探 → 高级侦探 → 侦探大师 → 传奇侦探
  • 经验值获取，完成案件获得积分
  • 成就系统，解锁专属徽章
👥

历史人物图鉴

  • 收集各个时期的历史名人
  • 了解他们的成就和贡献
  • 技能标签展示人物特点

技术栈

React 19 TypeScript Vite 6 Framer Motion Zustand Lucide React clsx

快速开始

# 安装依赖

# 构建生产版本
pnpm build

src/
├── components/

游戏设计理念

🎮 游戏化学习

通过探案、解谜的方式学习历史知识

🔓 渐进式解锁

完成当前任务才能解锁新区域，保持玩家动力

🏆 成就系统

通过收集历史人物、解锁成就建立成就感

🎨 中国风设计

使用中国传统色彩和元素，营造沉浸感

历史时期设计

时期 时代特点 代表人物 重要事件
先秦 诸子百家 孔子、老子 百家争鸣
秦汉 大一统 秦始皇 统一六国
三国 英雄辈出 刘备、曹操、诸葛亮 赤壁之战
隋唐 盛世繁华 唐太宗 贞观之治
宋元 科技文化 毕昇、岳飞 四大发明
明清 封建末期 郑和 下西洋
