Project Overview

History Detective Squad is an educational adventure game themed around Chinese history. Players take on the role of a young "history detective," traveling through different historical periods, solving mysteries, answering questions, and collecting historical character cards to become true history masters.

History Detective Squad Educational Adventure Game

Key Features

🗺️

Historical Time Map

  • Six major historical periods: Pre-Qin, Qin-Han, Three Kingdoms, Sui-Tang, Song-Yuan, Ming-Qing
  • Fog unlock mechanism - unlock new areas by completing investigation tasks
  • Exquisite Chinese-style visual design
🔍

Investigation Mission System

  • Multiple investigation tasks for each era
  • Questions covering historical figures, major events, cultural achievements
  • Difficulty levels: Easy, Medium, Hard
👤

Detective Training System

  • Level progression: Apprentice → Junior → Senior → Master → Legendary Detective
  • Experience points - earn points by completing cases
  • Achievement system - unlock exclusive badges
👥

Historical Figure Collection

  • Collect historical figures from each period
  • Learn about their achievements and contributions
  • Skill tags showcase character traits

Quick Start

# Build for production
